# Банкетам.Нет — вариант 4 (ГИА 09.02.07)

Портал бронирования помещений для банкетов. PHP + MySQL (XAMPP), экран 390×844.
Статусы заявки: Новая → Банкет назначен → Банкет завершен.

## Запуск
1. Скопировать папку в `C:\xampp\htdocs\`.
2. В XAMPP запустить **Apache** и **MySQL**.
3. phpMyAdmin → «Импорт» → `schema.sql` (создаст базу `banket_net`).
4. Открыть `http://localhost/<имя-папки>/`.

- Администратор: `Admin26` / `Demo20`

## Перенос на другой вариант (≈5 минут)
1. `assets/slide1.jpg…slide4.jpg`, `assets/logo.png` — заменить файлы (имена не менять).
2. `fonts/main.ttf` — положить выданный шрифт.
3. `css/theme.css` — HEX палитры в `:root`.
4. `theme.php` — тексты, сущность, статусы, список позиций, `db_name`.
5. `schema.sql` — имя базы данных в первых строках.
