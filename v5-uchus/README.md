# Учусь.РФ — вариант 5 (ГИА 09.02.07)

Портал записи на онлайн-курсы. PHP + MySQL (XAMPP), мобильный экран 390×844.

## Запуск
1. Скопировать папку в `C:\xampp\htdocs\`.
2. Запустить в XAMPP **Apache** и **MySQL**.
3. Открыть **phpMyAdmin** → вкладка «Импорт» → выбрать `schema.sql` → выполнить.
   (создаст базу `uchus_rf` со всеми таблицами и данными)
4. Открыть `http://localhost/<имя-папки>/`.

## Доступы
- Тестовый пользователь: `test12` / `test1234`
- Администратор: `Admin26` / `Demo20` (страница `admin.php`)

## Перенос на другой вариант (≈5 минут)
1. Картинки слайдера: заменить файлы `assets/slide1.jpg … slide4.jpg`, `assets/logo.png` (имена не менять).
2. Шрифт: положить выданный шрифт в `fonts/main.ttf` (или поправить путь в `css/theme.css`).
3. Палитра: поменять HEX в `:root` в `css/theme.css`.
4. Тексты/сущности/статусы/курсы: `theme.php` (блок «ЗАМЕНИ ПОД ВАРИАНТ»).
5. Имя базы данных: `theme.php` (`db_name`) + первые строки `schema.sql`.
