# Single-file версии (один файл на вариант)

Тот же портал, но **весь проект в одном `.php` файле**: можно открыть файл, Ctrl+A → копировать
целиком. Внутри по секциям, разделённым комментариями:

- `ВАРИАНТ` — название/сущность/статусы/позиции + палитра (точка замены под вариант)
- `MYSQL` — схема и **автосоздание базы при первом открытии** (импорт в phpMyAdmin не нужен)
- `PHP` — логика (сессии, роутер, обработка форм, ООП-класс `Db`)
- `CSS` — оформление (палитра, мобильный экран 390×844)
- `JS` — слайдер

## Файлы
- `passenger.php` — Пассажирам.РФ (вариант 3)
- `vodit.php` — Водить.РФ (вариант 1)
- `conference.php` — Конференции.РФ (вариант 2)
- `banket.php` — Банкетам.Нет (вариант 4)
- `uchus.php` — Учусь.РФ (вариант 5)

## Запуск
1. Положить нужный файл в `C:\xampp\htdocs\` (например `index.php`).
2. В XAMPP запустить **Apache** и **MySQL**.
3. Открыть `http://localhost/index.php` — база и таблицы создадутся сами, с тестовыми данными.

Доступы: пользователь `test12` / `test1234`, админ `Admin26` / `Demo20`.

> Роутинг — через `?page=index|register|login|cabinet|application|admin|logout`.
> Под новый вариант на экзамене: поменять блок `ВАРИАНТ` (тексты/статусы/позиции) и палитру `$PAL` вверху файла.
